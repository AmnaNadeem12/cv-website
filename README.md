# Personal CV Website

## Project Information
This is a responsive personal CV website built using HTML, CSS, and Flexbox. It includes profile, education, skills, projects, and contact sections with dark/light mode,contact form and a downloadable CV.

## Technologies Used
- HTML5
- CSS3 (Flexbox)
- JavaScript
